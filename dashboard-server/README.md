# Dashboard Server

## Описание

Бэкенд для поддержки клиентов, созданный на основе стартового репозитория TypeScript фреймворка Nest

## Установка

```bash
$ npm install
```

## Запуск приложения

```bash
# режим разработки
$ npm run start

# режим отслеживания изменений
$ npm run start:dev

# продакшн режим
$ npm run start:build
```

## Тестирование

```bash
# модульные тесты
$ npm run test

# end-to-end тесты
$ npm run test:e2e

# проверка покрытия тестами
$ npm run test:cov
```
